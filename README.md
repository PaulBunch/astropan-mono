# AstroPan Mono

**AstroPan Mono** — специализированный иконочный шрифт для астрологических интерфейсов. Разработан для обеспечения предельной четкости на малых размерах экрана при строгом соблюдении моноширинной сетки.

Проект вдохновлен инженерными решениями **JetBrains Mono**, перенося их на специфику астрологической символики.

---

## Ключевые особенности

  * **Pixel Perfect (10px):** Геометрия оптимизирована так, что при кегле 10px основной штрих (100 UPM) занимает ровно 1 физический пиксель. Это исключает «размытие» на стандартных мониторах (96 dpi).
  * **Умная моноширинность (Smart Grid):**
      * **Base (600 UPM):** Стандартный модуль для большинства символов.
      * **Wide (900/1200 UPM):** «Полуторная» или «двойная» ширина для плотных глифов (например, Плутон, Дева). Это позволяет сохранить читаемые просветы внутри символа, не жертвуя толщиной штриха.
  * **Строгая сетка:** Все узлы, толщины и радиусы скруглений привязаны к сетке в 10 единиц, что гарантирует чистоту форм при рендеринге.

## Технические характеристики

| Параметр               | Значение             |
| :--------------------- | :------------------- |
| **Units Per Em (UPM)** | 1000                 |
| **Main Stroke**        | 100 UPM (1px @ 10px) |
| **Thin Stroke**        | 60 UPM               |
| **Target Size**        | 10px                 |
| **Форматы**            | `.woff2`, `.ttf`     |

---

## Состав шрифта (v1.0)

Набор включает **39 базовых символов**:

  * **Зодиак (12):** Стандартные Unicode-позиции.
  * **Планеты (10):** Основной септенер + высшие планеты.
  * **Аспекты (5):** Мажорные аспекты (0°, 60°, 90°, 120°, 180°).
  * **Дома (12):** Расположены в зоне **PUA** (`U+E000`–`U+E00B`).

## Использование

Микротипографика в web (CSS):

```css
@font-face {
  font-family: 'AstroPan Mono';
  src: url('dist/astropan-mono.woff2') format('woff2');
  font-weight: normal;
  font-style: normal;
  font-display: block;
}

.astro-icon {
  font-family: 'AstroPan Mono', monospace;
  font-size: 10px;
  line-height: 1;
  -webkit-font-smoothing: antialiased;
  -moz-osx-font-smoothing: grayscale;
}
```

## Разработка и сборка

Проект использует автоматизированный пайплайн сборки из исходных SVG-файлов.

### Структура проекта

  * `src/glyphs/` — исходные векторы каждого символа.
  * `scripts/build.py` — основной пайплайн сборки (требует `fonttools`).
  * `docs/spec.md` — полная техническая спецификация проекта.

### Сборка

```bash
# Установка зависимостей
pip install fonttools brotli

# Компиляция шрифта
python scripts/build.py
```
