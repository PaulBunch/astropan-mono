# AstroPan Mono

**AstroPan Mono** — специализированный иконочный шрифт для астрологических интерфейсов. Разработан для обеспечения предельной четкости на малых размерах экрана при строгом соблюдении моноширинной сетки.

Проект вдохновлен инженерными решениями **JetBrains Mono**, перенося их на специфику астрологической символики.

---

## Ключевые особенности

  * **Grid-Aware Geometry (10px):** Геометрия знаков спроектирована с учетом 10-пиксельной сетки (шаг 100 UPM). 
  * **Main Stroke (80 UPM):** Основной штрих 80 UPM центрирован по осям пикселей. Данная величина выбрана как математически оптимальная для современных алгоритмов растеризации. Она обеспечивает четкость (1 физический пиксель) при системном масштабировании 125% и гарантирует симметричное сглаживание на экранах 96 dpi / 144 dpi, сохраняя при этом максимальный внутренний просвет (counter-space) в сложных глифах.
  * **Умная моноширинность (Smart Grid):**
      * **Base (600 UPM):** Стандартный модуль для большинства символов.
      * **Wide (900/1200 UPM):** «Полуторная» или «двойная» ширина для плотных глифов (например, Плутон, Дева). Это позволяет сохранить читаемые просветы внутри символа, не жертвуя толщиной штриха.
  * **Строгая сетка:** Все узлы, толщины и радиусы скруглений привязаны к сетке в 10 единиц, что гарантирует чистоту форм при рендеринге.

## Технические характеристики

| Параметр               | Значение         |
| :--------------------- | :--------------- |
| **Units Per Em (UPM)** | 1000             |
| **Main Stroke**        | 80 UPM           |
| **Thin Stroke**        | 50 UPM           |
| **Target Size**        | 10px             |
| **Форматы**            | `.woff2`, `.ttf` |

---

## Состав шрифта (v1.0)

Набор включает **39 базовых символов**:

  * **Зодиак (12):** Стандартные Unicode-позиции.
  * **Планеты (10):** Основной септенер + высшие планеты.
  * **Аспекты (5):** Мажорные аспекты (0°, 60°, 90°, 120°, 180°).
  * **Дома (12):** Расположены в зоне **PUA** (`U+E000`–`U+E00B`).

## Использование

Микротипографика в web (CSS):

```css
@font-face {
  font-family: 'AstroPan Mono';
  src: url('/dist/astropan-mono.woff2') format('woff2');
  font-weight: normal;
  font-style: normal;
  font-display: block;
}

.astro-icon {
  font-family: 'AstroPan Mono', monospace;
  font-size: 10px;
  line-height: 1;
  -webkit-font-smoothing: antialiased;
  -moz-osx-font-smoothing: grayscale;
}
```

## Разработка и сборка

Проект использует автоматизированный пайплайн: кастомные скрипты преобразуют **SVG-исходники** в формат **UFO**, после чего шрифт компилируется через **fontmake**.

### Структура проекта

  * `src/glyphs/` — исходные SVG-векторы символов.
  * `src/config.toml` — параметры шрифта (именование, метрики) отдельно от графики.
  * `scripts/build.py` — основной скрипт сборки.
  * `docs/spec.md` — техническая спецификация.

### Сборка

Требуется **Python 3.11+** (используется стандартная библиотека `tomllib`).

Скрипт автоматически импортирует векторы из Inkscape, нормализует кривые и генерирует файлы шрифтов:

```bash
# Установка зависимостей
pip install fontmake ufo2ft fonttools brotli

# Запуск сборки
python scripts/build.py
```
