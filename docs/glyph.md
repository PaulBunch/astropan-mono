# Инструкция по подготовке глифов в Inkscape

Для корректной сборки шрифта каждый символ должен быть представлен в виде **единого замкнутого контура**. «Живые» фигуры (круги, прямоугольники) или пересекающиеся линии компилятор может проигнорировать или обработать с ошибками.

## Правила организации слоев
Для удобства работы и автоматизации рекомендуется разделять «черновик» и «финал»:

* **construction** — рабочий слой. Здесь вы рисуете примитивами, используете обводки (Stroke) и накладываете фигуры друг на друга. Этот слой игнорируется скриптом сборки.
* **glyph** — слой-артефакт. Сюда помещается итоговый результат подготовки. Скрипт `build.py` ищет данные только в этом слое.

---

## Алгоритм подготовки (Check-list)
Если вы закончили отрисовку символа в слое `construction`, выполните следующие шаги для создания финального контура:

1. **Дублирование**: Выделите все объекты в слое `construction` и нажмите `Ctrl + D`.
2. **Оконтуривание** (Object/Stroke to Path):
  * Нажмите `Ctrl + Alt + C`, чтобы превратить толщину линий (Stroke) в контуры.
  * Нажмите `Shift + Ctrl + C`, чтобы превратить примитивы (прямоугольники, эллипсы) в кривые Безье.
3. **Слияние** (Union): Нажмите `Ctrl + +` (Сумма), чтобы сплавить все пересекающиеся части в один объект. Это исключит появление «дырок» и артефактов при масштабировании.
4. **Перенос**: Переместите полученный объект в слой `glyph`.
5. **Очистка** (Optional):
  1. Проверьте узлы: удалите лишние точки вручную.
  2. Не используйте `Ctrl + L` (Simplify) без необходимости, так как это может исказить мелкие детали геометрии.
   
## Технические требования к финальному контуру

* **Тип данных**: Тег `<path>` с атрибутом `d="..."`.
* **Заливка**: Только сплошной цвет (Fill), без градиентов и обводок.
* **Замкнутость**: Контур обязан быть замкнутым.
